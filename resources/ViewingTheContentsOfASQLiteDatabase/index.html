Viewing the Contents of a SQLite Database

  1. Start the emulator or plug in physical device
  2. In Android Studio, navigate to View > Tool Windows > Device File Explorer
  3. In the panel that displays, navigate to the data > data directory
  4. Find your package and expand it (ex. com.example.taylorherald.favoriteurlslivedata)
  5. Open up the databases directory within that package, right click on the .db file (not the journal) and save
  6. At this point, there are quite a few options to choose from (browser add-ons, terminal on Mac, standalone apps, ...); I prefer to use a browser plugin as it seems to be the easiest (I also haven't shopped around a whole lot, so if you find anything that seems pretty slick be sure to share!)
  7. On Firefox or Google Chrome, add either the SQLite Manager or SQLite Reader extension. The directions below will be for using SQLite Reader.
  8. Open up the SQLite Reader extension and at the top of the window click to browse to your .db file saved in step 5 and select it
  9. At first load, a query will execute showing the different tables in the database. At this point you are free to play around!
