Studio 2: Web APIs

Prompt:

The goal of this studio is simple: find an appropriate web API from the link in the resources and apply it to the Open Air Quality Template project. You should not only pull data from a different web API, but also create new models for any objects there might be. Note: You do not have to pull all pieces of data that are returned in the JSON from a web API; just choose a handful that you believe are relevant.

One API that I found fairly easy to work with was Open Library.
